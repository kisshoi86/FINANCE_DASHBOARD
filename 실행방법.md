# F&F 실적 대시보드 실행 방법

## ✅ 설치 완료!

필요한 패키지가 설치되었습니다. 이제 대시보드를 실행할 수 있습니다.

## 🚀 실행 방법

### 방법 1: Python 모듈로 실행 (권장)

프로젝트 폴더에서 PowerShell을 열고:

```powershell
python -m streamlit run dashboard.py
```

### 방법 2: Streamlit 명령어 직접 실행

```powershell
streamlit run dashboard.py
```

### 방법 3: 배치 파일 사용

`run_dashboard.bat` 파일을 더블클릭하거나:

```powershell
.\run_dashboard.bat
```

## 📍 현재 위치 확인

대시보드를 실행하기 전에 올바른 폴더에 있는지 확인하세요:

```powershell

pwd

# 프로젝트 폴더로 이동 (필요한 경우)
cd "C:\Users\AC1160\OneDrive - F&F\바탕 화면\finance_dashboard"
```

## 🌐 브라우저 접속

실행 후 자동으로 브라우저가 열립니다. 수동으로 열려면:

- **기본 주소**: http://localhost:8501
- 다른 포트 사용: `streamlit run dashboard.py --server.port 8502`
